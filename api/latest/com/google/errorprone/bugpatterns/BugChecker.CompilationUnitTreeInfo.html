Class BugChecker.CompilationUnitTreeInfo

    public abstract static class BugChecker.CompilationUnitTreeInfo
extends Object
    The information that is safe for a BugChecker.CompilationUnitTreeMatcher to access.

    Error-prone does not support matching entire compilation unit trees, due to a limitation of javac. Class declarations must be inspected one at a time via BugChecker.ClassTreeMatcher.

    CAUTION: checks can still access the compilation unit tree using VisitorState.getPath(), but the AST nodes for type declarations may be in an inconsistent state.
